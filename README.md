## rl.notes

Модуль для «1С-Битрикс: Управление сайтом», реализующий бэкенд и фронтенд для сервиса заметок.

Демо: http://notes.red---line.ru/ (http-авторизация notes/notespass)

# Установка модуля
1. Скопируйте модуль в директорию /bitrix/modues/
2. В административной части перейдите в раздел Marketplace -> Установленные решения, и в контекстном меню модуля "Сервис заметок" выберите пункт "Установить".
3. На нужной странице публичной части сайта разместите компонент rl:notes.
Пример вызова компонента
```php
<?$APPLICATION->IncludeComponent(
    "rl:notes",
    "",
    []
);?>
```

##Описание установки
При установке модуля:
1. Создается тип инфоблоков notes и инфоблок notes для хранения заметок
2. API копируется в директорию /api/notes/
3. В файле /urlrewrite.php создаются правила для маршрутизации запросов к API
4. Компонент rl:notes копируется в директорию /local/components/

#Некоторые детали реализации
1. API предлагает все CRUD-операции и работает с инфоблоком notes (точка входа /api/notes/)
2. Авторизация имитируется через жестко прописанный токен
3. Фронтенд-компонент реализован псевдо-компонентом на BitrixVue
